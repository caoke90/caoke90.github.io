# 快速开始

如何使用vue组件container，



## 引入js

```html
<script src=https://cdn.jsdelivr.net/npm/vue@2.6.10/dist/vue.js></script>
<script type=text/javascript src=js/demo.d6af508.js></script>
```

## 页面放入组件容器

```html
<div id=app>
  <mv-modal></mv-modal>
  <container :mkey=></container>
</div>
```



## 获取字典数据

`const dataMap=JSON.parse(localStorage.getItem('demoData'));`

## 设置字典数据

`Vue.Container.setDataMap(dataMap);`

## vue渲染

 ``//渲染`
  `new Vue({`
    `el: '#app',`
    `data:{`
      `home:dataMap._demo`
    `}`
  });`



## demo

```html
<!DOCTYPE html>
<html>

<head>
  <meta charset=utf-8>
  <meta content="initial-scale=1,minimum-scale=1,maximum-scale=1,user-scalable=no,width=device-width,viewport-fit=cover,telephone=yes" name=viewport>
  <title>demo</title>
  
</head>

<body>

<div id=app>
  <mv-modal></mv-modal>
  <container :mkey=home></container>
</div>

<script src=https://cdn.jsdelivr.net/npm/vue@2.6.10/dist/vue.js></script>
<script type=text/javascript src=js/demo.d6af508.js></script>
<script>

  //获取数据
  const dataMap=JSON.parse(localStorage.getItem('demoData'));

  //给容器设置数据
  Vue.Container.setDataMap(dataMap);

  //渲染
  new Vue({
    el: '#app',
    data:{
      home:dataMap._demo
    }
  });
</script>
</body>

</html>
```
