第3节：容器-字典

容器数据是由 容器名mkey和数据数组 组成的数据格式

字典={
    //页面组件
    page1:{
        组件
    }，
    //home容器
    home:[
        {
            组件
        },{
            组件
        }
    ],
    //shop容器
    shop:[
        {
            组件
     },{
            组件
        }
    ],
    //person容器
    person:[
        {
            组件
        }
    ]，
}

在页面中使用

•字典[mkey] 为对象时： container为页面组件

•字典[mkey] 为数组时：container为容器

<container mkey='home'></container>
<container mkey='shop'></container>
<container mkey='person'></container>
<container mkey='box'></container>
