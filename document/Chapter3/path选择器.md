# 第8节：path选择器

容器的数据结构是key-value形式的对象类型的数据，（非多叉树结构的），所以path选择器最多只有两级。



## 当前选中：home>0

第一级是容器名（mkey字符类型）,第二级是容器内的组件位置（数字类型）

## 用途

1. 查询，用于组件的修改
2. 定位位置，用于组件的添加、删除

## 容器数据模型

```
字典={
	//页面组件
	page1:{
        "card_type": "box",
        "title": '标题',
        "mkey": 'home',
  	},
    //home容器
	home:[
        组件,
        组件，
        组件，
    ],
   
}
```
